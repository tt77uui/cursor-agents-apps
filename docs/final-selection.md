# 最终选定：主推 APP

## 结论

**主推（唯一上线打磨目标）**：`app-01-shop-quote-studio` — **报价单工坊**

## 对比与理由

| 维度 | App01 报价单工坊 | App02 排班库存轻助手 |
|------|------------------|----------------------|
| 付费冲动 | 高：导出/Pro 模板直接省时间 | 中：偏内部工具 |
| 交付复杂度 | 低：模板 + 计数 + 解锁 | 中：多实体 CRUD + 周视图 |
| 合规叙事 | 清晰：模板草稿 + 非法律意见 | 低敏感 |
| 冷启动推广 | 易：维修/家政/个体户社群 | 较难：需触达店长 |

App02 作为**备选仓库内保留**，便于后续扩展「多门店」订阅；根命令 `npm run start:prod` 仅启动 App01 以降低运维面。

## 归档说明

未删除 App02 代码（便于对比与复用组件）；生产单应用部署请只用 **App01 的 Dockerfile** 或 Compose 中的 `app01` 服务。
