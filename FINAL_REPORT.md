# 最终报告 — 主推：报价单工坊（app-01-shop-quote-studio）

## 功能介绍

- **模板驱动**：内置「维修报价」「简易收据」等字段化模板；Pro 含「服务套餐报价」模板。
- **本地列表**：报价单标题、模板、字段值持久化在 SQLite。
- **导出**：一键导出最新一条为纯文本（含免责声明），非 Pro 有每日次数上限。
- **解锁**：输入解锁码即可本地标记 Pro（演示用；生产对接已授权支付后发码）。
- **响应式 UI**：移动端优先的单页布局；非侵入式横幅广告位。

## 盈利分析

| 模式 | 适配度 | 说明 |
|------|--------|------|
| 基础免费 + Pro | 高 | 模板包/导出次数是强付费点，客诉低 |
| 按导出次数 | 中 | 可与 Pro 叠加，适合偶发大单用户 |
| 小额订阅 | 中 | 适合高频门店，需支付通道 |
| 横幅广告 | 低-中 | 作为补充，勿用弹窗与违规追踪 |

**客单价建议**：一次性模板包 ¥9.9–29；或 ¥6/月 低频订阅（需合规支付）。

## 目标用户

- 上门维修、家政保洁、手机维修、小批发档口店主。
- 习惯微信成交、需要快速「看起来像样」的文本摘要，而非复杂 ERP。

## 上线推广建议

1. **垂直社群**：同城维修群、家政派单群，提供免费模板 + 截图对比。
2. **短视频**：15 秒演示「30 秒出报价单」；评论区放官网（非医疗法律承诺）。
3. **异业合作**：与配件店、打印店互推（横幅位可换本地合作）。
4. **SEO 长尾**：「维修 报价单 模板 文本」等长尾词落地页。
5. **合规页**：单独「免责声明」页面链接放在导出页脚与商店描述。

## 运维与交付

- 一键开发：`npm run start`（根目录同时起两应用）；**生产主推**：`npm run start:prod`（仅 App01）。
- 容器：见 `deploy/docker-compose.yml` 与各自 `Dockerfile`。
